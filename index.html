We believe that data science should be data driven,
with a strong focus on types. Our aim is to bring powerful tools
to the best language for this task in our opinion, Haskell.

  • Interested in Data + Haskell? Make it real!

Data is important, we cannot do data science without data.
Haskell preserves the integrity of it.

Haskell is a strongly typed pure functional language. You can't go wrong.
Functional programming is more suited for data science than the imperative paradigm.
Your program becomes a data transformation pipeline, based on functions and expressions that are composable.
Also, Haskell is fast!
Contribution oportunity for all data science or Haskell skill levels.

dataHaskell is an open source organization devoted to improve the environment of data science in Haskell.

We focus on being completely open source, purely functional, documentation bringers, friendly, integrators that assure our environment without conflicts with other libraries and passionate about sharing ideas, code and projects.
